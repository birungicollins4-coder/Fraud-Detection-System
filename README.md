# Financial Fraud Detection System

## Project Overview

This machine learning project detects fraudulent financial transactions based on transaction patterns and behavior.

---

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib

---

## Features

- Fraud classification
- Transaction pattern analysis
- Risk prediction
- Fraud visualization

---

## Results

- Accuracy: 94.6%
- High fraud detection precision
- Reduced false positives
